字符串处理

字符串处理函数是面试中必考也是日常我们业务逻辑开发中常用到的，因此我们在面试之前一定要将常用的字符串处理函数牢记于心
for example：
先简单来一个简单的小例子来练练手：（业务场景：隐藏掉手机号的中间几位数字，用*代替）

$str = 1852271352;
echo substr_replace($str,'***',3,3);   //output:185***1351

设想一下如果你不使用函数去做处理是一种都没痛苦的事情

living example one

$str = 'Uploads/a.b.php';  
尽可能多的获取.php或者php

考点：
1）字符串的处理

分析：
1）这是一个很经典的PHP的面试题目，考察的点都是PHP的一些函数，包含了字符处理函数，数组处理函数，文件处理函数

下面我先从字符串处理函数来进行解答这道题
1）substr函数

substr($str,-3);

2）使用strrpos和substr函数配合使用

substr($str,strrpos($str,'.'));

3）使用strrchr函数

strrchr($str,'.');

4）使用数组函数：explode和array_pop

$arr = explode('.',$str);
echo array_pop($arr);

5）使用文件系统函数pathinfo

pathinfo($str)['extension']

living example two

不用PHP函数实现字符串反转

//数组越界的问题
Notice: Uninitialized string offset: 7 in D:\wamp\www\phpTest\test.php on line 7  
$str = 'abcdefg';
$len = strlen($str);
$return = '';
for($i=$len-1;$i>=0;$i--){
    $return.=$str{$i};
}
echo $return;

living example three

请写一个函数，实现以下功能： 字符串“open_door” 转换成 “OpenDoor”、
”make_by_id” 转换成 ”MakeById”

function o($str){
    $arr = explode('_',$str);
    $str = '';
    foreach($arr as $v){
        $str.=ucfirst($v);
    }
    return $str;
}
echo o('make_by_id');

分析：
two和three两道题都是考察我们php的字符串处理函数
