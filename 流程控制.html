1、switch....case...语法的一个小问题

$a = 1;
switch ($a) {
    case $a >= 0:
        echo 0;
        break;
    case $a >= 10:
        echo 1;
        break;

    default:
        echo 2;
        break;
}
//ouput:0

分析：

原因：$a 非0即是true 所以要找到下面是true的结果，那么也就是第一个了，下面同理是

转换成一个false 那么下面哪个是false呢，那就是$a >= 10

$a = 0;  
switch ($a) {
    case $a >= 0:
        echo 0;
        break;
    case $a >= 10:
        echo 1;
        break;

    default:
        echo 2;
        break;
}

分析：

输出1

因此不要随便使用switch去做比较，要做的就是我传的这个值是否想匹配，那么就不会出现这个问题了
